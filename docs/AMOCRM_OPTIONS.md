# 🎯 Дополнительные опции amoCRM

## Обзор

В калькулятор добавлена возможность подключения двух дополнительных опций amoCRM:

1. **Файловое пространство** (доступно на всех тарифах)
2. **Супер-поля** (доступно только на тарифах Growth и Enterprise)

---

## 💾 Файловое пространство

### Описание
Расширенное хранилище файлов для всех тарифов amoCRM.

### Стоимость
**999 ₽/мес** × период использования (с учетом бонусов)

### Доступность
✅ Доступно на **всех тарифах**:
- Start (Базовый)
- Growth (Расширенный)
- Enterprise (Профессиональный)

### Примеры расчета

#### Пример 1: Период 6 месяцев
```
Стоимость: 999 ₽/мес
Период оплаты: 6 мес
Период использования: 7 мес (6 + 1 от партнера)
Итого: 999 × 7 = 6 993 ₽
```

#### Пример 2: Период 12 месяцев
```
Стоимость: 999 ₽/мес
Период оплаты: 10 мес
Период использования: 13 мес (10 + 2 от amoCRM + 1 от партнера)
Итого: 999 × 13 = 12 987 ₽
```

#### Пример 3: Период 24 месяца
```
Стоимость: 999 ₽/мес
Период оплаты: 18 мес
Период использования: 25 мес (18 + 6 от amoCRM + 1 от партнера)
Итого: 999 × 25 = 24 975 ₽
```

---

## ⚡ Супер-поля

### Описание
Расширенные поля для тарифов Growth и Enterprise, позволяющие создавать более сложные структуры данных.

### Стоимость
**999 ₽/мес** × период использования (с учетом бонусов)

### Доступность
❌ **НЕ доступно** на тарифе Start (Базовый)  
✅ **Доступно** на тарифах:
- Growth (Расширенный)
- Enterprise (Профессиональный)

### Автоматическое отключение
При переключении с Growth/Enterprise на Start тариф, опция "Супер-поля" автоматически отключается.

### Примеры расчета

Аналогично "Файловому пространству":
- **6 мес**: 999 × 7 = 6 993 ₽
- **12 мес**: 999 × 13 = 12 987 ₽
- **24 мес**: 999 × 25 = 24 975 ₽

## 💰 Расчет стоимости

### Формула
```
Стоимость опции = 999 ₽/мес × Итоговый период использования
```

### Важно!
Используется **итоговый период с бонусами**, а не период оплаты:
- 6 мес → 7 мес
- 12 мес → 13 мес
- 24 мес → 25 мес

### Пример полного расчета

**Конфигурация:**
- Тариф: Growth (1 199 ₽/мес)
- Пользователей: 5
- Период: 12 мес
- Опции: Файловое пространство + Супер-поля

**Расчет:**
```
Лицензии:
  1 199 ₽/мес × 5 польз × 10 мес = 59 950 ₽
  Период использования: 13 мес

Файловое пространство:
  999 ₽/мес × 13 мес = 12 987 ₽

Супер-поля:
  999 ₽/мес × 13 мес = 12 987 ₽

────────────────────────────────────────
Итого лицензии и опции: 85 924 ₽
```

## 🔄 Логика работы

### При включении опции
1. Пользователь отмечает чекбокс
2. Рассчитывается стоимость: 999 × `totalDurationLicense`
3. Отображается итоговая сумма под чекбоксом
4. Добавляется в общую стоимость проекта

### При выключении опции
1. Пользователь снимает чекбокс
2. Стоимость обнуляется
3. Удаляется из общей стоимости

### При смене тарифа на Start
1. Если была включена опция "Супер-поля"
2. Она автоматически отключается
3. Показывается сообщение "Требуется Growth или Enterprise"

### При смене периода
1. Автоматически пересчитывается `totalDurationLicense`
2. Обновляется стоимость опций
3. Обновляется отображение "Итого: X ₽ (Y мес)"

## 📝 Технические детали

### Состояние
```typescript
const [amoCrmOptions, setAmoCrmOptions] = useState({
  superFields: false,   // Супер-поля
  fileStorage: false,   // Файловое пространство
});
```

### Расчет стоимости
```typescript
let amoCrmOptionsTotal = 0;
if (crm === 'amocrm') {
  const totalDuration = getTotalDuration(period, crm);
  if (amoCrmOptions.superFields) amoCrmOptionsTotal += 999 * totalDuration;
  if (amoCrmOptions.fileStorage) amoCrmOptionsTotal += 999 * totalDuration;
}
```

### Автоотключение супер-полей
```typescript
useEffect(() => {
  if (licenseTariff === 'start' && amoCrmOptions.superFields) {
    setAmoCrmOptions(prev => ({ ...prev, superFields: false }));
  }
}, [licenseTariff]);
```
