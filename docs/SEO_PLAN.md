## Структура контента (помимо текущих страниц)

### Раздел `/blog` (создать в фазе 2)
Кластеры статей:
1. **«Гайды по amoCRM»** — настройка воронок, интеграции, секреты

## Технический SEO — чеклист

### Уже сделано в Next-проекте
- Server-side рендеринг — поисковики видят полный HTML
- Metadata API на каждой странице
- Динамический sitemap.xml
- robots.txt
- Hreflang (RU/EN) в alternates
- JSON-LD Schema.org (Organization, WebSite, Service, Breadcrumb)
- Canonical URLs
- Безопасные заголовки (X-Content-Type-Options и пр.)

### Сделать после деплоя
- [ ] Google Search Console — подтвердить домен, отправить sitemap
- [ ] Яндекс.Вебмастер — подтвердить домен, отправить sitemap
- [ ] Google Analytics 4 + Яндекс.Метрика
- [ ] Vercel Speed Insights
- [ ] Core Web Vitals: LCP < 2.5s, INP < 200ms, CLS < 0.1
- [ ] Открытые OG-картинки (`/og-image.jpg`)
- [ ] Все `<img>` → `next/image` с явными `width`/`height` и `alt`
- [ ] Inline критический CSS Tailwind подтянет автоматом
- [ ] Lighthouse 90+

## Что не делать

- Не покупать дешёвые ссылки на биржах — фильтры Яндекса
- Не публиковать AI-контент без редактуры — Google Helpful Content Update наказывает
- Не делать дубли страниц без canonical
