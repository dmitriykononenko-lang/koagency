# Обновление тарифной системы Kommo: Переход на доллары США

## Дата обновления
29 января 2026

## Основные изменения

### 1. Конвертация валюты для Kommo
Все расчеты для Kommo теперь производятся в **долларах США** (USD):
- Курс конвертации: **1 USD = 90 RUB**
- Добавлена функция `convertToUSD()` для автоматической конвертации цен

### 2. Бесплатные мессенджеры для Kommo
**Facebook Lead** и **Messenger** теперь **бесплатны** при использовании Kommo:
- Базовая абонентская плата: 0 USD
- Настройка остается платной (в долларах при выборе Kommo)

### 3. Обновленная логика ценообразования

#### Функция calculatePrice()
- Все цены пакетов внедрения конвертируются в USD для Kommo
- Лицензия Kommo Advanced остается в USD (фиксированные пакеты)
- Абонентская плата за мессенджеры конвертируется в USD
- Дополнения (1С, Roistat, Документы) конвертируются в USD

#### Функция getLineItems()
- Добавлена вспомогательная функция `priceConvert()`
- Все позиции внедрения автоматически конвертируются для Kommo:
  - Базовый пакет: 60 000 ₽ → ~667 USD
  - Бизнес пакет: 90 000 ₽ → ~1 000 USD
  - Кастом пакет: от 150 000 ₽ → ~1 667 USD

#### Функция getSubItems()
- Добавлена вспомогательная функция `subPriceConvert()`
- Facebook Lead и Messenger отображаются как бесплатные для Kommo
- Все остальные мессенджеры конвертируются в USD
- Avito конвертируется с учетом базовой платы и дополнительных аккаунтов

### 4. Функция getMessengerPrice()
Обновлена логика для бесплатных мессенджеров:
```typescript
if (['facebook_lead', 'messenger'].includes(service) && crm === 'kommo') {
  baseMonthly = 0; // Бесплатно для Kommo
}
```

## Технические детали

### Константы
```typescript
const USD_RATE = 90; // 1 USD = 90 RUB
```

### Вспомогательные функции
```typescript
// Конвертация рублей в доллары с округлением
const convertToUSD = (rubAmount: number) => {
  return Math.round(rubAmount / USD_RATE);
};
```

## Обратная совместимость

### amoCRM
Все цены для amoCRM остаются в рублях:
- Лицензии: 599/1199/1699 ₽ за пользователя
- Мессенджеры: оплачиваются по обычным тарифам в рублях
- Пакеты внедрения: в рублях

### Kommo
- Лицензии Advanced: в долларах (фиксированные пакеты)
- Все остальные расчеты: конвертация из рублей в доллары
- Facebook Lead и Messenger: бесплатно
